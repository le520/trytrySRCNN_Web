# trytrySRCNN_WEB
## 项目说明
该项目为trytrySRCNN的WEB体验版本,基于Django开源框架。

trytrySRCNN链接：[trytrySRCNN](https://github.com/heiheishuang/trytrySRCNN)

## 使用说明

将权重文件放在/static/pths/目录下并将文件名更改为best.pth即可。（当然你也可以在代码中修改这一名称）
